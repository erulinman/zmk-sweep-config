# Инструкции для работы с раскладкой клавиатуры

## Отображение раскладки

После каждого изменения файла `config/cradio.keymap` ОБЯЗАТЕЛЬНО показывай актуальную раскладку ВСЕХ слоёв в виде ASCII-схемы клавиатуры в следующем формате:

```
## 🔤 Слой 0 - Основной (QWERTY + Home Row Mods)

╭─────────────────────────────────────────────╮   ╭─────────────────────────────────────────────╮
│                 ЛЕВАЯ ПОЛОВИНА              │   │                ПРАВАЯ ПОЛОВИНА              │
╰─────────────────────────────────────────────╯   ╰─────────────────────────────────────────────╯

┌───────┬───────┬───────┬───────┬───────┐       ┌───────┬───────┬───────┬───────┬───────┐
│   Q   │   W   │   E   │   R   │   T   │       │   Y   │   U   │   I   │   O   │   P   │
└───────┴───────┴───────┴───────┴───────┘       └───────┴───────┴───────┴───────┴───────┘
┌───────┬───────┬───────┬───────┬───────┐       ┌───────┬───────┬───────┬───────┬───────┐
│   A   │   S   │   D   │   F   │   G   │       │   H   │   J   │   K   │   L   │   ;   │
│ SHIFT │ CTRL  │  ALT  │  GUI  │       │       │       │  GUI  │  ALT  │ CTRL  │ SHIFT │
└───────┴───────┴───────┴───────┴───────┘       └───────┴───────┴───────┴───────┴───────┘
┌───────┬───────┬───────┬───────┬───────┐       ┌───────┬───────┬───────┬───────┬───────┐
│   Z   │   X   │   C   │   V   │   B   │       │   N   │   M   │   ,   │   .   │   /   │
└───────┴───────┴───────┴───────┴───────┘       └───────┴───────┴───────┴───────┴───────┘
                    ┌───────┬───────┐       ┌───────┬───────┐
```

## Требования:

1. **Показывай ВСЕ слои** - основной (0), правый (1), левый (2), три-слой (3)
2. **Используй эмодзи** для заголовков слоёв:
   - 🔤 для основного слоя
   - 🔢 для слоя с цифрами
   - 🔣 для слоя с символами
   - ⚙️ для системного/три-слоя
3. **Добавляй описание** каждого слоя (как активируется)
4. **Показывай Home Row Mods** второй строкой для клавиш с модификаторами
5. **Отмечай активные клавиши слоя** символом ▓▓▓▓▓ в thumb-кластере
6. **Используй понятные обозначения** для специальных клавиш:
   - Стрелки: ←, ↓, ↑, →
   - Медиа: ⏮, ⏯, ⏭
   - BT0/1/2 для Bluetooth профилей
   - VOL+/VOL-/MUTE для громкости

## Легенда

Всегда включай легенду в конце с объяснением обозначений:
- `MO1/MO2` - Momentary layer (зажать для активации слоя)
- `▓▓▓▓▓` - Активная клавиша слоя
- `BT0/1/2` - Bluetooth профили
- `BT CLR` - Очистить Bluetooth соединения
- `BOOT` - Bootloader mode
- Вторая строка в клавишах - модификаторы при удержании (Home Row Mods)
